# TheJoester's FoundryVTT Macros

This is a collection of macros that work with FoundryVTT. 

Welcome to the foundryvtt-macros wiki!

## Foundry Macros

[**Advanced Pull to Scene**](https://github.com/thejoester/foundryvtt-macros/wiki/Advanced-Pull-to-Scene) - Select Scene from dropdown, select users, send to that scene.

[**Bulk Set Scene config**](https://github.com/thejoester/foundryvtt-macros/wiki/Bulk-Set-Scene-Config) - Change scene configurations in bulk. 

[**CSV to Roll Table**](https://github.com/thejoester/foundryvtt-macros/wiki/CSV-to-Roll-Table) - Upload a .csv file into a roll table

[**List Modules**](https://github.com/thejoester/foundryvtt-macros/wiki/List-Modules) - List all modules, with version, and website link if available, sorted by enabled / disabled. 

[**Send Whisper**](https://github.com/thejoester/foundryvtt-macros/wiki/Send-Whisper) - UI to send whisper to user

[**Sorted RollTable Preview**](https://github.com/thejoester/foundryvtt-macros/wiki/RollTable-Sorted-Preview) - preview the contents of any RollTable in the world, sorted alphabetically

[**Toggle Lights**](https://github.com/thejoester/foundryvtt-macros/wiki/Toggle-lights) - Toggle lights on and off.

[**World Data Report**](https://github.com/thejoester/foundryvtt-macros/wiki/%5BFoundryVTT%5D-World-Data-Report) - Show report of world data sizes used by items (scenes/actors/items/etc.)

## PF2e Macros

[**Award XP**](https://github.com/thejoester/foundryvtt-macros/wiki/%5BPF2e%5D-Award-XP) - Award XP based on selected Tokens

[**Award Hero Points**](https://github.com/thejoester/foundryvtt-macros/wiki/%5BPF2e%5D-Award-Hero-Points) - Award or set Hero Points for all actors in party, send message to chat.

[**Calculate Encounter Difficulty & XP**](https://github.com/thejoester/foundryvtt-macros/wiki/%5BPF2e%5D-Calculate-Encounter-Difficulty-&-XP) - Calculate Encounter Difficulty from Selected Tokens.

[**Everlight Crystal Toggle**](https://github.com/thejoester/foundryvtt-macros/wiki/%5BPF2e%5D-Everlight-Crystal-Toggle) - This macro toggles the equipped state of the "Everlight Crystal".

[**Group Roll**](https://github.com/thejoester/foundryvtt-macros/wiki/%5BPF2e%5D-Group-Roll) - (GM) Roll Check/Save for multiple selected tokens (Characters or NPCs) at once. 

[**Light Spell Orb**](https://github.com/thejoester/foundryvtt-macros/wiki/%5BPF2e%5D-Light-Spell-Orb) - Macro to create token for light spell orb, player can choose color. 

[**Make Loot Pile**](https://github.com/thejoester/foundryvtt-macros/wiki/Make-Loot-Pile) - (GM) Create single loot pile from multiple selected NPC tokens. 

[**Party Recall Knowledge**](https://github.com/thejoester/foundryvtt-macros/wiki/%5BPF2e%5D-Party-Recall-Knowledge) - (GM) Roll Recall Knowledge for all party members, save to Journal.

[**Prompt Class DC Save**](https://github.com/thejoester/foundryvtt-macros/wiki/%5BPF2e%5D-Prompt-Class-DC-Save) - Player or GM can prompt target token to make save (Fortitude / Reflex / Will) in chat message, DC is selected token's Class DC

[**Search Actors**](https://github.com/thejoester/foundryvtt-macros/wiki/%5BPF2e%5D-Search-Actors) - (GM) Search selected tokens, shows report of items, spells, featc, etc. 

[**Show XP**](https://github.com/thejoester/foundryvtt-macros/wiki/%5BPF2e%5D-Show-XP) - Sends message to Chat showing all party actors and their current XP

[**Sort Inventory**](https://github.com/thejoester/foundryvtt-macros/wiki/%5BPF2e%5D-Sort-inventory) - Sorts the selected token inventory.
